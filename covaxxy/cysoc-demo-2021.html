CoVaxxy Data

Real-time data collection

- Constantly growing

- 120M+ tweets

- Two virtual machines ensure we have no data outages

  • Employ a snowball sampling technique to create a list of over 70 keywords related to covid-19 and vaccines
  • There are many, however, over 95% of our tweets so far include the first three words highlighted in red
  • We then set up a real=time Twitter stream that downloads any tweet containing any of these keywords - beginning on January 4th
  • This means we have:
    • Constanty growing
    • 120M+ tweets
    • To ensure no gaps in data, we’ve set up two virtual machines in different geographical locations that collect the exact same data
